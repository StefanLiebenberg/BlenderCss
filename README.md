BlenderCss
==========

BlenderCss provides a java toolset that compiles css from different supersets of css together.



## Maven


```xml
<dependency>
 <groupId>com.google.closure-stylesheets</groupId>
 <artifactId>blender-css</artifactId>
 <version>1.0</version>
 </dependency>
```
